# Journal Writing Skill: Quarterly Journal of Economics

## Journal Profile
- **ABS 2024 Rating:** 4*
- **Field:** ECON
- **Publisher:** Oxford University Press (on behalf of President and Fellows of Harvard College)
- **Scope:** General-interest economics across all fields; empirical, theoretical, and policy research. Founded 1886.
- **Guidelines URL:** https://academic.oup.com/qje/pages/General_Instructions
- **Field baseline:** econ-field-writing-style
- **Articles spot-checked:** 2 (Jäger, Roth, Roussille, Schoefer 2024, DOI 10.1093/qje/qjae001; Bouscasse, Nakamura, Steinsson 2025, DOI 10.1093/qje/qjae046)

## Hard Constraints (Author Guidelines)

> These override FIELD_SKILL where they conflict.

### Structure
Submission-phase assembly order is prescribed: Title page (abstract and acknowledgment footnote on this page) → Text → Author affiliations (immediately after last line of text, not on title page) → Appendix → References → Notes → Tables → Figures. Each table on a separate page. ENFORCED.

### Length
No explicit word or page limit stated in guidelines.

### Abstract
Unstructured; maximum 250 words. Citations are explicitly prohibited ("reference citations must be avoided"). JEL codes must appear at the end of the abstract. Both articles place JEL codes inline as the final sentence (e.g., "JEL Codes: J01, J31, J42, J64, D83."). ENFORCED.

### Citations
Author-year parenthetical. Follows FIELD_SKILL norm.

### Headings
Guidelines specify headings "should be to journal style." Journal style, confirmed across both articles: main section headings use Roman numerals followed by ALL CAPS titles (e.g., "I. INTRODUCTION", "II. ANCHORED BELIEFS ABOUT OUTSIDE OPTIONS"). FIELD_SKILL states Roman or Arabic are both acceptable; QJE enforces Roman numerals. ENFORCED.

### Tables and Figures
Tables numbered consecutively using Roman numerals (Table I, Table II). ENFORCED per guidelines. Figures follow the same Roman numeral scheme by practice (Figure I, Figure II). PRACTICE ONLY for figures.

### Explicit Prohibitions
Citations must not appear in the abstract. ENFORCED.

## Observed Deltas from FIELD_SKILL

> Only dimensions where this journal deviates from the field baseline.

### Structural Deviations

**Heading numbering:** FIELD_SKILL states "Roman or Arabic numerals; either is acceptable." QJE enforces Roman numerals in ALL CAPS for main section headings. Subsections follow a Roman+letter scheme (II.A., II.B., II.C.) with title case; sub-subsections use Arabic numerals with a period (1., 2., 3.) with title case. The subsection and sub-subsection schemes are not specified in guidelines but are consistent across both articles. Main heading style: ENFORCED. Subsection scheme: PRACTICE ONLY.

**Acknowledgment placement:** Acknowledgments appear as a footnote on the first page, attached to the article title by an asterisk. There is no standalone Acknowledgments section. FIELD_SKILL does not address acknowledgment placement. PRACTICE ONLY (guidelines reference "acknowledgment footnote" on title page).

**Author affiliations:** Placed directly after the last line of text, not on a title page or cover page. ENFORCED.

### Other Journal-Specific Patterns

**Data replication requirement:** After acceptance, authors are required to post data, programs, and a readme PDF to the QJE Dataverse (hosted on Harvard Dataverse). This is a post-acceptance mandatory deposit. ENFORCED.

**Abstract JEL codes:** JEL codes are not a separate field but appear as the final sentence of the abstract itself. Both articles confirm this placement. ENFORCED.

**Footnotes:** Numbered consecutively; guidelines specify they should be "kept as brief as possible." Footnotes appear at the bottom of the relevant page in published articles. ENFORCED.

## Common Pitfalls

1. **Citations in the abstract:** QJE explicitly prohibits reference citations in the abstract. This is unusual enough to be a submission error in practice.
2. **Arabic section numbers:** Using "1.", "2." for main sections instead of "I.", "II." conflicts with enforced journal style. Subsections require the compound Roman+letter format (II.A.), not standalone letters or Arabic numbers.
3. **Forgetting JEL codes in the abstract body:** JEL codes must appear as the last sentence of the abstract itself, not in a separate field. Omitting them or placing them elsewhere conflicts with guidelines.
4. **Replication package:** QJE requires deposit to Harvard Dataverse on acceptance. Failure to prepare a complete replication package (data, code, readme PDF) before submission can delay publication.
