## Hard Constraints (Author Guidelines)

> These override FIELD_SKILL where they conflict.

### Structure
Follows FIELD_SKILL norm for standard empirical and theoretical article types. No journal-specific deviations noted in guidelines.

**Label:** FIELD DEFAULT

### Length
Initial submissions should strive to be **less than 50 pages** of double-spaced 12-point text with 1-inch margins, **all-inclusive** (includes everything: title page, abstract, references, tables, figures, appendices). Alternative: **40 pages in 11-point font**. Papers far shorter than the limit are welcomed.

**Label:** ENFORCED

### Abstract
- **Maximum 250 words**
- **Citations strongly discouraged**. If a citation is essential, the author may be queried to remove it or recast the sentence. If the author insists the reference be cited, the **entire reference must be cited in square brackets** within the abstract (not just author-date format).
- Format (structured vs. unstructured): not specified; both acceptable per FIELD_SKILL norm.

**Label:** ENFORCED

### Citations
**INFORMS citation style** (NOT APA, despite similarity):
- In-text: author-date format (Norman 1977) or (Norman 1977)
- Reference list: alphabetical by author name, formatted per INFORMS style guide
- While similar to APA in using author-date format, this is technically INFORMS house style and should be cited as such.

**Label:** ENFORCED

### Formatting
- **Double-spaced** with **1-inch margins all around**
- **Font:** Times Roman or Garamond (or similar). **Explicitly prohibited:** Helvetica Narrow.
- **Submission format:** PDF or MS Word

**Label:** ENFORCED

### Headings
- May be **numbered or unnumbered** (author's choice)
- If numbered, follow the number with a period (e.g., "1. Introduction", "2. Literature Review")
- Must be **consistent** within a given article
- INFORMS journals recommend using **one or two levels of headings**

**Label:** ENFORCED

### Tables and Figures

**Tables:**
- Numbered with Arabic numerals
- Must have a title
- Column headings should be **brief** and **avoid abbreviations**
- **INFORMS house style requires only three horizontal rules**: (1) above table column heads, (2) between column heads and table body, (3) after table body. No vertical rules.
- If table is reproduced exactly from a source: "Source: ___"
- If adapted or created from someone else's data: "Adapted from ___"

**Figures:**
- Each figure must be numbered and have a caption
- Captions should be **complete sentences** that explain the figure fully without requiring the reader to consult the text
- If the caption exceeds one sentence: break and set second and subsequent sentences as a lightface note below figure, preceded by heading **"Note"** (for one sentence) or **"Notes"** (for multiple sentences)
- Original images must be supplied with **300 dpi resolution**, will be converted to CMYK or grayscale

**Label:** ENFORCED

### Explicit Prohibitions
- **Helvetica Narrow font** (specifically called out)
- **Citations in abstracts** (strongly discouraged; if essential, must provide full reference in square brackets)
- **Simultaneous submission** to other journals (may submit to conferences simultaneously)
- **Non-original material, copyrighted material, or previously published material** in refereed venues

**Label:** ENFORCED

### Review Process
- **Double-blind review required**: authors must remove names, institutions, and acknowledgments from submitted manuscript
- ORCID iD required for submitting author
- Compliance certification required: Code of Ethics, Data and Methods Transparency Policy
- If revisions requested: resubmit within three months minimum, no later than one year

**Label:** ENFORCED

### Citation Style and Reference Format

**FIELD_SKILL norm:** "APA in-text citations: (Author, Year) or Author (Year)"

**Organization Science practice:** Uses **INFORMS citation style**, which employs author-date format similar to APA but is technically distinct. In-text citations: (Norman 1977) or Norman (1977). References listed alphabetically. While functionally similar to APA for basic citations, authors should format references according to INFORMS style guide specifications, not APA manual.

**Label:** ENFORCED (stated in guidelines)

### Abstract Citations

**FIELD_SKILL norm:** "No citations in abstract"

**Organization Science practice:** Citations **strongly discouraged** but not absolutely prohibited. If essential, author may be queried to remove or recast. If author insists, **full reference must be cited in square brackets** (not just author-date). This is more specific than FIELD_SKILL's blanket prohibition.

**Label:** ENFORCED (stated in guidelines)

### Manuscript Length Specification

**FIELD_SKILL norm:** "Typical length 10,000–15,000 words" for quantitative deductive work

**Organization Science practice:** Explicit page limit of **<50 pages all-inclusive** (double-spaced, 12pt, 1" margins) or **<40 pages in 11pt font**. "All-inclusive" means title page, abstract, main text, references, tables, figures, appendices—everything. Papers "far shorter" than the limit are welcomed. This is stricter and more explicit than FIELD_SKILL.

**Label:** ENFORCED (stated in guidelines)

### Font Restrictions

**FIELD_SKILL norm:** No specific font guidance

**Organization Science practice:** Recommends Times Roman or Garamond. **Explicitly prohibits Helvetica Narrow** (unusual specificity). This suggests journal has strong readability preferences.

**Label:** ENFORCED (stated in guidelines)

### Table Formatting

**FIELD_SKILL norm:** Tables use standard academic formatting

**Organization Science practice:** **INFORMS house style requires exactly three horizontal rules** in tables: above column heads, between heads and body, after body. No vertical rules. Column headings must be brief and avoid abbreviations. More prescriptive than FIELD_SKILL.

**Label:** ENFORCED (stated in INFORMS style guide)

### Figure Captions

**FIELD_SKILL norm:** "Captions for figures and tables should be written as complete sentences"

**Organization Science practice:** If caption exceeds one sentence, subsequent sentences are set as **lightface note below the figure**, preceded by **"Note"** (singular) or **"Notes"** (plural) heading. This is a specific formatting rule not mentioned in FIELD_SKILL.

**Label:** ENFORCED (stated in INFORMS style guide)

### Heading Numbering

**FIELD_SKILL norm:** No specific guidance on heading numbering

**Organization Science practice:** Authors may choose numbered or unnumbered headings, but must be **consistent throughout**. If numbered, use period after number (e.g., "1. Introduction"). Recommends limiting to one or two heading levels. This gives authors more flexibility than some journals.

**Label:** ENFORCED (stated in INFORMS style guide)

## Common Pitfalls

Based on author guidelines, common submission mistakes specific to Organization Science:

1. **Exceeding the 50-page limit** including references, tables, and appendices. Authors often miscalculate by counting only main text. The limit is all-inclusive.

2. **Using APA reference format instead of INFORMS style**. While both use author-date citations, INFORMS has its own formatting specifications that differ from APA in details.

3. **Including citations in the abstract** using standard (Author Year) format. If citations are essential, the full reference must be provided in square brackets.

4. **Table formatting with multiple horizontal rules or vertical rules**. INFORMS house style permits only three horizontal rules; additional formatting will be removed in production.

5. **Failing to remove author identifiers for double-blind review**. Common oversights: acknowledgments, self-citations that reveal identity, file metadata, funding statements that name investigators.

6. **Using Helvetica Narrow font**, which is explicitly prohibited for legibility reasons.
