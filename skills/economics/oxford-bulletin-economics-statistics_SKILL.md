### Structure
Follows FIELD_SKILL norm.

### Length
Not determinable from available sources.

### Abstract
Unstructured paragraph. JEL Classification codes appear inside the abstract box (not as a separate header field). Keywords listed above the abstract box, separated by pipe characters. No word limit confirmed. — PRACTICE ONLY

### Citations
**Major deviation.** OBES uses numbered citations in square brackets [1], [2], [3], with a sequentially numbered reference list at the end. This directly supersedes FIELD_SKILL, which states author-year is near-universal. In-text citations are always bracketed integers; no author names appear in-text unless introduced in running prose. — PRACTICE ONLY (guidelines inaccessible; confirmed consistently throughout published article)

### Headings
Arabic numerals with pipe separator: "1 | Introduction", "2.1 | Subsection Title", "3.1.1 | Sub-subsection Title". Three levels of numbered heading are used. Tables and figures follow the same convention: "TABLE 3 | Title text" and "FIGURE 1 | Caption text". — PRACTICE ONLY

### Tables and Figures
Follows FIELD_SKILL norm for content (standard errors in parentheses, significance stars with ***1%, **5%, *10% noted in table notes). Formatting uses "TABLE X |" and "FIGURE X |" label convention. — PRACTICE ONLY

## Observed Deltas from FIELD_SKILL

### Citation Style
**Field norm:** Author-year parenthetical or in-text is near-universal; numbered footnote references are not the norm.
**OBES practice:** Numbered square-bracket citations [X] throughout, with a sequentially numbered bibliography. Running prose never embeds "(Author Year)"; instead references are "[X]" only. This is the single most consequential submission-prep difference.
**Label:** PRACTICE ONLY (guidelines inaccessible; confirmed throughout full published article)

### Endnotes vs. Footnotes
**Field norm:** Footnotes carry supplementary material and qualifications.
**OBES practice:** Numbered endnotes collected in a dedicated "Endnotes" section immediately before the reference list, not at page bottoms. Content function is the same (qualifications, data notes, robustness caveats), but placement differs.
**Label:** PRACTICE ONLY

### Abstract Format
**Field norm:** FIELD_SKILL does not specify abstract layout details.
**OBES practice:** JEL codes are embedded within the abstract box (not a separate metadata field). Keywords appear above the abstract box as a distinct labeled line. Both are required elements.
**Label:** PRACTICE ONLY

## Common Pitfalls
1. **Submitting with author-year citations.** OBES uses numbered references [X]. Submitting a draft formatted with "(Author Year)" will require a full citation-style conversion before or at acceptance.
2. **Omitting JEL codes.** JEL codes appear in the abstract box in published articles and are almost certainly required at submission; do not treat them as optional.
3. **Using footnotes.** OBES collects notes as endnotes; write notes expecting end-of-paper placement, not page-bottom placement.
4. **Confusing ABS tier with top-journal norms.** As an ABS 3 journal, OBES publishes shorter empirical papers (the checked article is 17 published pages) and does not require the exhaustive robustness battery of ABS 4*/4 outlets—though robustness checks remain expected.
